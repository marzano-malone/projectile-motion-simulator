# Projectile Motion Simulator

This project models projectile motion using Python.

## Description
The program allows the user to input velocity and launch angle, and calculates:
- Time in air
- Maximum height
- Horizontal range

## Why I made this
I created this project to explore how mathematical concepts from Applied Mathematics can be used to model real-world motion.

## What I learned
- How to translate mathematical formulas into code
- How changing variables affects motion
- How programming can simulate physical systems

## Technologies Used
- Python
