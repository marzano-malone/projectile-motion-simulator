

 # Projectile Motion Simulator

I'm a student developer interested in Engineering, currently building projects to explore Python and its real-world applications.

## The Problem
Calculating the motion of a projectile assuming no air resistance or drag.

## The Solution
I used Python to model projectile motion. Its powerful libraries, such as matplotlib, allow visualisation of trajectories without needing to build everything from scratch.

## Features
- Calculates trajectory based on velocity and angle
- Computes:
- Time in air
- Maximum height
- Horizontal range
- Displays a graph of the motion

## The Math
This project is based on standard kinematic equations for projectile motion.

## Why I Made This
I created this project to explore how mathematical concepts from Applied Mathematics can be used to model real-world motion.

## What I Learned
- How to translate mathematical formulas into code
- How changing variables affects motion
- How programming can simulate physical systems

## Technologies Used
- Python
- matplotlib

## Example Output
Enter velocity (m/s): 20 Enter angle (degrees): 45

Time in air: 2.89 seconds Range: 40.82 metres Maximum height: 10.20 metres

## How to Run
1. Install dependencies:
pip install matplotlib

2. Run the program:
python3 projectile.py
